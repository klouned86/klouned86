 2. Вывести адрес приложения todolist в системе Android

Вывести все приложения телефона 

C:\Users\klouned86\Downloads\platform-tools_r31.0.3-windows\platform-tools>adb shell pm list packages -f

Адрес приложения todolist в системе Android

C:\Users\klouned86\Downloads\platform-tools_r31.0.3-windows\platform-tools>adb shell "pm list packages -f | grep todolist"

 3. Установить .apk файл приложениия todolist на телефон с компьютера через  ADB

C:\Users\klouned86\Downloads\platform-tools_r31.0.3-windows\platform-tools>.\adb install -t "D:\todo.apk"

Success

 4. Сделать скриншот запущенного приложения todolist и сразу скопировать на компьютер в одной команде.

C:\Users\klouned86\Downloads\platform-tools_r31.0.3-windows\platform-tools>adb shell screencap /storage/emulated/0/Pictures/screen_1.png && adb pull /storage/emulated/0/Pictures/screen_1.png D:\QA

 5. Вывести в консоль логи приложения todolist

C:\Users\klouned86\Downloads\platform-tools_r31.0.3-windows\platform-tools>adb logcat | findstr com.android.todolist


 6. Скопировать логи приложения todolist на компьютер.

C:\Users\klouned86\Downloads\platform-tools_r31.0.3-windows\platform-tools>adb logcat >> D:\QA\log.txt


 7. Удалить приложение todolist с телефона через ADB

C:\Users\klouned86\Downloads\platform-tools_r31.0.3-windows\platform-tools>adb uninstall com.android.todolist
